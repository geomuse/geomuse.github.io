# CMD

```bash
CMD touch /home/hello/hello.txt
```

# EXPOSE 

用于设置与主机相连的端口号

```bash
EXPOSE 80
EXPOSE 443
```

# ENV

环境变数

```bash
ENV HELLO 1234
CMD echo $HELLO
```

也可以使用 `-e` 设置环境变数

```
sudo docker run -e  HELLO=4321 example
```
