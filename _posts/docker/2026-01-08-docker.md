# cp

```bash
docker cp hello-nginx:/etc/nginx.conf ./
```

命令解释

执行上述命令，将nginx.conf文件复制到当前目录

```bash
docker cp <容器名称>:<路径> <主机路径>
```

# bash

| 语法 | 说明
| -- | :--: | 
| > | 输出重定向 
| < | 输入重定向 
| >> | > 添加到尾端
| 2> | 将命令执行的标准错误保存为文件
| 2>> | 将命令执行的标准错误添加到文件
| &> | 将标准输出和错误都保存为文件
