# pull

从docker hub下载镜像

```
sudo docker pull ubuntu:latest
```

镜像名称中 / 为之前指定的使用者

# build

```
sudo docker build --tag  hello:0.1 .
```

# run

```
sudo docker run --name hello-nginx -d -p 80:80 -v /root/data:/data hello:0.1
```
