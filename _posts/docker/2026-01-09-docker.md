# FROM 

```bash
FROM scratch
FROM ubuntu:14.04   
```

一切从`FROM`开始，不然不能创建镜像

创建镜像时,要在Dockerfile 所在目录使用 `docker build`

```bash
sudo docker build --tag example . 
```

`--tag` 是设置镜像名词
