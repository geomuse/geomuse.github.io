# Java 软件开发 30 天学习计划

## **Day 1-5：Java 基础**
### **学习目标**
- 理解 Java 的核心语法。
- 掌握变量、数据类型、运算符、条件语句、循环等基本编程概念。
- 学习类与对象的基础知识。

### **学习内容**
1. **Java 环境配置**
   - 安装 JDK（Java Development Kit）
   - 配置环境变量
   - 使用 IntelliJ IDEA 或 Eclipse 作为 IDE

2. **Java 基本语法**
   - Hello World 程序
   - 变量和数据类型
   - 操作符和表达式

3. **条件语句和循环**
   - if-else 语句
   - switch 语句
   - for 循环、while 循环、do-while 循环

4. **方法（函数）和数组**
   - 定义方法
   - 方法的参数和返回值
   - 一维数组和多维数组

5. **类与对象**
   - 定义类和创建对象
   - 类的属性和方法
   - 构造方法

### **练习项目**
1. **简单计算器**
   - 功能：实现加、减、乘、除运算。
   - 要求：用户输入两个数字和一个运算符，输出结果。

2. **通讯录管理小程序**
   - 功能：增加、删除、查询联系人信息。
   - 要求：使用数组或 ArrayList 存储联系人。

---

## **Day 6-10：Java 面向对象编程（OOP）**
### **学习目标**
- 理解面向对象编程的四大核心概念：封装、继承、多态、抽象。
- 学习接口和内部类的使用。
- 掌握异常处理和文件读写操作。

### **学习内容**
1. **封装、继承、多态**
   - 什么是封装？
   - 继承的基本用法
   - 多态的概念和实现

2. **接口和内部类**
   - 定义接口
   - 接口的实现
   - 匿名内部类和局部内部类

3. **异常处理**
   - try-catch 语句
   - 自定义异常
   - finally 关键字

4. **文件读写（I/O）**
   - File 类的使用
   - BufferedReader 和 BufferedWriter
   - 序列化和反序列化

### **练习项目**
1. **用户管理系统**
   - 功能：用户的增删改查功能。
   - 要求：使用面向对象编程的思想设计系统。

2. **文本编辑器**
   - 功能：打开、编辑和保存文件。
   - 要求：支持基本的文本操作功能。

---

## **Day 11-15：GUI 应用开发**
### **学习目标**
- 学习 Java 的图形用户界面开发。
- 使用 JavaFX 或 Swing 创建图形化桌面应用程序。

### **学习内容**
1. **JavaFX 项目结构**
   - 创建 JavaFX 项目
   - 理解 JavaFX 的基本架构

2. **布局管理器**
   - BorderPane、VBox、HBox、GridPane
   - Scene 和 Stage 的使用

3. **基本组件**
   - 按钮、文本框、标签、菜单栏等
   - 事件处理机制

### **练习项目**
1. **图形化计算器**
   - 功能：实现加、减、乘、除运算。
   - 要求：使用 JavaFX 或 Swing 开发。

2. **记事本软件**
   - 功能：支持打开、编辑和保存文本文件。
   - 要求：添加菜单栏和工具栏。

---

## **Day 16-20：数据库开发**
### **学习目标**
- 学习如何使用 JDBC 连接数据库。
- 掌握 SQL 的基本操作。
- 学习使用 DAO（数据访问对象）模式进行数据库操作。

### **学习内容**
1. **JDBC 基础**
   - 加载数据库驱动
   - 连接数据库
   - 执行 SQL 语句

2. **SQL 基本操作**
   - 创建表、插入数据
   - 查询、更新、删除数据

3. **DAO 模式**
   - 什么是 DAO？
   - 创建 DAO 类和接口
   - 使用 DAO 访问数据库

### **练习项目**
1. **库存管理系统**
   - 功能：管理商品信息。
   - 要求：数据存储在 SQLite 或 MySQL 数据库中。

2. **日程管理软件**
   - 功能：记录个人日程安排。
   - 要求：支持添加、修改和删除日程。

---

## **Day 21-25：多线程与文件操作**
### **学习目标**
- 理解多线程编程的概念。
- 学习如何创建和管理线程。
- 学习文件加密/解密的基本操作。

### **学习内容**
1. **多线程基础**
   - 创建线程的两种方式
   - 线程的生命周期
   - 线程同步和锁机制

2. **文件操作**
   - 文件读写
   - 文件加密/解密

3. **网络通信**
   - 套接字编程
   - 基本的客户端/服务器模型

### **练习项目**
1. **下载管理器**
   - 功能：支持多线程下载文件。
   - 要求：显示下载进度。

2. **文件加密工具**
   - 功能：对文本文件进行加密和解密。
   - 要求：支持密码保护。

---

## **Day 26-30：软件打包与部署**
### **学习目标**
- 学习如何将 Java 应用程序打包成可执行文件。
- 创建软件安装包。
- 掌握软件的版本控制和更新机制。

### **学习内容**
1. **打包成 JAR 文件**
   - 使用命令行打包
   - 使用 IDE 打包

2. **创建安装程序**
   - 使用 Inno Setup 创建 EXE 安装包
   - 创建 Mac 的 DMG 文件

3. **版本控制和更新**
   - 软件版本管理
   - 实现自动更新功能

### **练习项目**
1. **任务管理器**
   - 功能：管理用户的任务清单。
   - 要求：实现软件的更新机制。

2. **完整软件项目**
   - 将之前的项目打包成完整的软件包。
   - 创建安装程序并发布。
