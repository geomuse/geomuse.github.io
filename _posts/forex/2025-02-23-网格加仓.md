二、均值回归型补单（网格策略）

适用场景：震荡行情中降低成本

```c
// 示例代码：等差网格补单
int gridLevel = OrdersTotal();
double gridStep = 50*Point; // 50点间隔
if(gridLevel < 5) {
   double lot = 0.1*(gridLevel+1);
   OrderSend(Symbol(),OP_BUY,lot,Ask,3,0,0,"Grid_"+IntegerToString(gridLevel),Magic,0,Blue);
}
```

参数设置：

    网格层数：3-5层

    价格间隔：ATR(14)的1-1.5倍

    单层风险：≤0.5%

关键要点：

    必须设置总止损（如账户5%）

    配合波动率指标过滤

    禁止在单边市使用
