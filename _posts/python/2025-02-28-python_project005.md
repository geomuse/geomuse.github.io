任务：  
- 让用户输入一个字符串，统计其中：
  - 英文字母个数
  - 数字个数
  - 空格个数
  - 其他字符个数  

示例输入：  
```
请输入一串字符：Hello 123!
```
示例输出：  
```
字母个数：5
数字个数：3
空格个数：1
其他字符个数：1
```

提示： 使用 `isalpha()`、`isdigit()` 和 `isspace()` 方法判断字符类型。
