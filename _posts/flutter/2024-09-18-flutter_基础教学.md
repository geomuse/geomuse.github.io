在软体正中间建立 `I'm geo`

```dart
import 'package:flutter/material.dart';

main() {
  runApp(Center(
      child: Text(
    "I\'m geo",
    textDirection: TextDirection.ltr,
  )));
}
```

套用样式和设定文字

```dart
import 'package:flutter/material.dart';

main() {
  runApp(MaterialApp(
    home: Scaffold(
      appBar: AppBar(
        title: Text('Who am i'),
      ),
      body: Center(
        child: Text('I\'m geo'),
      ),
    ),
  ));
}
```
