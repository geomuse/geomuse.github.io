1. 什么是Flask？

    Flask是一个用Python编写的微型Web框架，具有简单灵活的特点。
    它适合开发小型应用或API，也能扩展用于大型项目。

2. Flask的特点和优点

    轻量级：核心很小，但功能强大。
    灵活性：可以根据需求自由选择扩展和库。
    模块化：易于集成其他Python库。

3. 安装和设置Flask开发环境

    安装Python和pip（Python包管理器）
    使用pip安装Flask：

```bash
pip install Flask
```

创建并激活虚拟环境：

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
venv\Scripts\activate  
```
